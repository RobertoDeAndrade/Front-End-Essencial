HTML Semântico


< header > - Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

< nav > - Uma seção com links de navegação.

< main > - Define o conteúdo principal dentro do body em seu documento ou aplicação.

< aside > - São os conteúdos laterais.

< article > - Podem ser posts, artigos, etc...

< footer > - É o rodapé.
