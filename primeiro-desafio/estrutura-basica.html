Estrutura básica do HTML


TAGS - São os elementos do HTML utilizados para informar ao navegador que tipo de estrutura está sendo construída. Iniciam-se com o sinal "<" e terminam com o sinal de ">".

< DOCTYPE html > - Indica a versão do HTML para o navegador.

< html > - Define o escopo de uma página HTML.

< head > - Informações de configuração.

< title > - Define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

< body > - Corpo do site, o que aparece na tela.

< h1 > a < h6 > - Tag para os títulos, sendo o h1 o nível mais alto e o h6 o mais baixo.

< p > - Tag para parágrafo.

< a > - Tag para, juntamente ao atributo href, criar uma hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

< img > - Tag para imagens.

< br > - Tag para quebra de linha.

< span > - Tag para um container genérico em linha para conteúdo fraseado, que não representa nada por natureza.
