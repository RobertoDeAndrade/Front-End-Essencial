Básico sobre estilos


CSS externo - Usamos através da tag < link > para selecionarmos e usarmos todos os estilos criados no arquivo CSS desejado.

Inline - É o estilo de CSS usado diretamente do arquivo HTML. Ele tem especificidade maior e sobrescreve outros estilos. Por boas práticas ele não é muito utilizado.

Estilização de texto - Para estilização de texto temos por exemplo o "color" para mudar a cor das letras, o "background-color" para alterar a cor de fundo do bloco, temos o "font-family" para escolhermos a fonte do texto, "font-style" para o estilo, como o itálico por exemplo e "font-weight" para o tamanho, além de diversas outras.

Box model - É a representação do elemento HTML (as tags) em um bloco retangular, que está dividido em 4 partes: conteúdo, espaçamento interno (padding), a borda (border) e a margem (margin).
